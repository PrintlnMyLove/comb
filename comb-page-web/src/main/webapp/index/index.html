关 于 我 们

蜂巢高校创业孵化器(简称蜂巢)是面向高校学生的科技创新基地. 我们发现了其他创业园平台的一些盲区, 并以此为切入点建立了蜂巢平台. 旨在提供高校学生和学校的创业园之间一个良性的互动平台. 鼓励大学生体验创业, 基类创业经验.

团队 & 用户
