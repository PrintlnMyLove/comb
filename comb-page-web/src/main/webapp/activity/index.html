创业达人

沈姵君

北京工商大学信息111

沈姵君给创业的小伙伴们发自肺腑的话：若你只是好奇，那就千万别创业；若你不是好奇，一旦创业就别停下来！记住：创业没有失败！

季朝阳

北京工商大学机械105

与他的谈话中，我可以感受到他对于创业的热情，但是他却十分想要告诉大家创业择人的重要性。创业热情高涨固然是好事，但是也要分析好自己，分析好市场，切忌盲目创业。创业是一条荆棘路，要从开始走的时候就做好心理准备。

杨鹏宇

北京工商大学广设112

所有想要创业的同学，一定要学会分析市场、了解市场。切忌盲目创业。学会找到市场需求。创业中的同学，得到父母的支持是十分重要的。而且对于专业课要好好学，对于一些商科管理类的课程，比如注册公司流程、经济法、税务、法人代表等可以选择选修或者自学。因为创业的成功是留给有准备的人的。
